01 — Fundamentos

Variables y Condicionales

Las variables son como cajas donde guardamos información. En JavaScript moderno usamos let para crear una variable. Los condicionales nos permiten tomar decisiones según el valor de esas variables.

Palabra clave Para qué sirve
let Crea una variable
if (condición) Ejecuta código solo si la condición es verdadera
{ } Bloque de código que se ejecuta si se cumple la condición
if … else { } Si no se cumple el if, se ejecuta lo que hay dentro del else
document.write() Muestra el resultado en pantalla

Ejemplo básico

let edad = 18;

if (edad >= 18) {
  document.write("Eres mayor de edad");
} else {
  document.write("Eres menor de edad");
}

02 — Repetición

Bucles FOR

Un bucle for repite un bloque de código un número determinado de veces. Tiene tres partes: inicio, condición y modificación.

ParteCódigoQué hace
iniciolet i = 0Prepara el contador
condicióni < 5Mientras sea true, se repite
modificacióni++Suma 1 al contador en cada vuelta

Estructura

for (inicio; condición; modificación) {
  // código que se repite
}

Ejemplo: mostrar números del 1 al 5

for (let i = 1; i <= 5; i++) {
  document.write(i + " ");
}
// Resultado: 1 2 3 4 5

Ejemplo: mostrar solo los números pares

for (let i = 0; i <= 10; i++) {
  if (i % 2 === 0) {
    document.write(i + " ");
  }
}
// Resultado: 0 2 4 6 8 10

03 — Organización

Funciones

Una función es un bloque de código con nombre que podemos reutilizar cuantas veces queramos. Primero se define y luego se llama (ejecuta).

AcciónCódigo
Definirfunction nombre() { }
Con parámetrofunction nombre(dato) { }
Llamarnombre();
Llamar con parámetronombre(valor);

Función sin parámetros

function saludar() {
  document.write("Hola, bienvenido");
}

saludar(); // Llamamos a la función

Función con parámetro

function saludar(nombre) {
  document.write("Hola, " + nombre);
}

saludar("Ana");  // Hola, Ana
saludar("Luis"); // Hola, Luis

04 — Estructura HTML

HTML y el DOM

El DOM (Document Object Model) es el árbol de elementos que forma una página web. JavaScript puede acceder a cualquier elemento del árbol y modificarlo.

Estructura de un árbol DOM básico

DOCUMENT └── html ├── head │ └── title "Mi Página" └── body ├── h1 "Bienvenidos" ├── p "Este es un párrafo" └── ul ├── li "Elemento 1" └── li "Elemento 2"

Código HTML equivalente

<!DOCTYPE html>
<html>
  <head>
    <title>Mi Página</title>
  </head>
  <body>
    <h1>Bienvenidos</h1>
    <p>Este es un párrafo</p>
    <ul>
      <li>Elemento 1</li>
      <li>Elemento 2</li>
    </ul>
  </body>
</html>

Seleccionar y modificar elementos

Podemos seleccionar elementos por su ID (devuelve uno solo) o por su clase (devuelve varios).

SelectorMétodoResultado

Ejemplo completo

// Por ID — modifica un único elemento
let titulo = document.getElementById("tituloPrincipal");
titulo.innerText = "Página Modificada";
titulo.style.color = "blue";

// Por Clase — modifica varios elementos
let parrafos = document.getElementsByClassName("texto");
parrafos[0].innerText = "Primer párrafo";
parrafos[1].innerText = "Segundo párrafo";
